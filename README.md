# Quadrupole Analysis

Python functions for taking molecular quadrupole tensors and converting to forms for comparison to literature. You can find the documentation for this package hosted on [ReadTheDocs](https://quadrupole.readthedocs.io/en/latest/index.html).

This package can be installed with any package manager that has access to the [Python Package Index](https://pypi.org/), such as `pip`. Development of this package is done using [uv](https://docs.astral.sh/uv/), and is recommended for most users. You can either install Quadrupole in a virtual environment:
```bash
> uv venv
...
> source .venv/bin/activate
> uv pip install quadrupole
```

or add it to a project:

```bash
> uv init
...
> uv add quadrupole
```
